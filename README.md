# BDD Hightlighter

Text highlighter for `Given`, `When`, `Then` and `And` keywords in VS Code.

VS Marketplace: https://marketplace.visualstudio.com/items?itemName=TamasSoos.bdd-highlighter

## Settings

```js
"BDDHighlighter": {
  "backgroundColor": "#1f171e",
  "color": "#e956ce",
  "borderRadius": "2px"
},
```
